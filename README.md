# Portfolio

This is my personal portfolio website built to showcase my work, skills, and experience as a developer.

The previous version of this repository contained a simple "Coming Soon" page. This release marks the first full public version of the portfolio (v1.0.0), featuring complete sections, live integrations, and production-ready deployment.

The focus of this project was clarity, performance, and simplicity.

---

## Live Website

Deployed via GitHub Pages.

---

## About

This portfolio is designed to be minimal and content-focused, with a strong emphasis on:

- Clean minimal dark UI  
- Smooth micro-interactions  
- Responsive layout  
- Performance optimization  
- Simple and readable structure  

No unnecessary libraries. No over-engineering.

---

## Features

- Fully responsive design  
- Modern dark interface  
- Smooth hover and transition effects  
- Project showcase section  
- Contact form integration (EmailJS)  
- Analytics tracking (GoatCounter)  
- Static deployment (GitHub Pages compatible)  

---

## Tech Stack

- **Next.js 14 (App Router)**  
- **TypeScript**  
- **Tailwind CSS**  
- **Lucide React**  
- **EmailJS** (client-side email handling)  
- **GoatCounter** (analytics)  

---

---

## Deployment

This project is configured for static export and works with GitHub Pages.

All integrations (analytics and email) are handled client-side, so no backend is required.
